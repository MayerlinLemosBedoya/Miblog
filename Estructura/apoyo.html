Apoya Biología Ilustrativa

Contribuye a mantener este proyecto de divulgación científica

Este proyecto nace del cruce entre la microbiología, el arte y la divulgación científica rigurosa.

Todo el contenido es gratuito y creado de forma independiente. Si este blog te ha servido para aprender, reflexionar o mirar la ciencia desde otro lugar, puedes apoyarlo voluntariamente.

¿Por qué apoyar este proyecto?

Biología Ilustrativa existe para:

  • → Democratizar el acceso a la ciencia de forma accesible y visual
  • → Crear ilustraciones científicas de calidad
  • → Conectar disciplinas: microbiología, diseño y educación
  • → Inspirar a nuevas generaciones de estudiantes de ciencia
